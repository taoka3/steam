# steam
steamのAPIを使用して積みゲーを調べてみた．

まず前提条件としてsteamのAPIを使用するにはsteamの商品をある一定の金額以上購入している人しか使えないみたい．
APIの登録は下記のリンクから可能です．

https://steamcommunity.com/dev/apikey

登録が終わったら登録したドメインから下記のソースコードを実行します．
なお、コンフィグ値はご自身にあった値へ変更してください．
